## 🔥 **HOT FEATURES - Dễ implement & High Impact**

### 🎰 **Slots Machine**
```
/slots [bet_amount]
🎰 | 🍒 | 🍒 | 🍒 |
     JACKPOT! 🎉
     Thắng: 50,000 xu
```
- 3-reel slots với symbols: 🍒🍋🍊🍇⭐💎
- Multiplier system: 3 giống = x10, 2 giống = x2
- Progressive jackpot tích lũy

### 🎲 **Dice Game (Tài Xỉu)**
```
/dice [bet] [tai/xiu]
🎲 Dice: 3 + 4 + 2 = 9 (XỈU)
✅ Bạn chọn XỈU - Thắng!
💰 Nhận: 20,000 xu
```
- Bet tài (11-18) hoặc xỉu (3-10)
- Payout 1:1, house edge thấp
- Animation với emoji

### 🃏 **Poker Hand**
```
/poker [bet]
🃏 Your hand: A♠ K♠ Q♠ J♠ 10♠
🎉 ROYAL FLUSH! 
💰 Thắng: 1,000,000 xu
```
- 5-card poker hands
- Payout theo độ hiếm: Pair x2, Flush x20, Royal Flush x1000
- Show probability

### 🎯 **Number Guessing Game**
```
/guess [number] [bet]
🎯 Số bí mật: 42
📍 Bạn đoán: 45 (Cách 3 số)
💰 Thắng: 5,000 xu (Gần đúng bonus)
```
- Đoán số 1-100
- Càng gần càng nhiều tiền
- Bonus nếu đúng chính xác
